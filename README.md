PracticalMachineLearning
========================

Practical Machine Learning course repository

This repository consists of the files:

- pml-training.csv
- pml-testing.csv

which are the files containing the training and test data used for this assignment,

- practicalmachinelearning.R
- practicalmachinelearning.Rmd
- practicalmachinelearning.html

which are the .R script, .Rmd document and HTML-document created for this assignment.
